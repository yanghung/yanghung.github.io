This dashboard allows the user to easily view supply chain metrics for 4 different vendors.

The main view lets the user verify validity of the shipment data sent by each vendor.

The invalid records are automatically flagged for follow-up with the vendors.

The user can then generate 10 different views of the valid records.

Tools used: Excel, VBA, SQL

There are 10 different buttons the user can click to see metrics in different "slices."

There are 7 different pre-formatted tables for the user to analyze. Each table is a different shipment metric.

The VBA code is optimized such that clicking different buttons will update all tables within 2-3 seconds.
